Telemachus README

This program is still being developed - and so is this documentation!

This program aims to be a graphical front for ShellMS on Android. Built for macOS, users can select a contact, then send an SMS to their number via their device.

Requires that ShellMS is installed on your android device. Grab it via F-Droid or from source here: https://github.com/try2codesecure/ShellMS

##Prerequisites##
- Have adb installed on your machine
- Have ShellMS installed on your android device

##Compilation##
1. Clone source
2. Open project in Xcode
3. Compile/run
